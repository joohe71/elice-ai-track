## 요약
---

- CSS
    - 한 태그의 고유한 CSS를 주고 싶을 때는 id를 사용
    - 범용성있는 CSS를 주고 싶을 때 클래스를 사용
    - 우선순위 적용 → **`id(#)`>`class(.)`>`tag`**
        - 같은 계열일 경우 아래쪽에 있는 것이 채택!
        - **`!important`** 붙으면 그 코드가 우선 적용
        - **`>`** 왼쪽의 화살표를 쓰면 해당 자식만 스타일 적용  
          화살표를 쓰지 않으면  모든 자식.손자 스타일 적용
